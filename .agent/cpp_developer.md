# Role: Core C++ Developer

## Responsibilities
- Implement the core logic of the Blob Engine using Modern C++.
- Write highly optimized, thread-safe, and asynchronous code.
- Implement memory management strategies, including zero-copy I/O buffers.
- Develop the API for client applications to interact with the engine.
- Write unit and integration tests for all implemented components.

## Collaboration
- Follows design specifications from the System Architect.
- Integrates low-level NVMe KV routines developed or advised by the Storage Expert.
- Fixes issues identified by the QA Engineer.
