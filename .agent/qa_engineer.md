# Role: QA & Performance Engineer

## Responsibilities
- Design and implement comprehensive test suites (unit, integration, and system tests).
- Develop benchmarking tools to measure IOPS, latency, and throughput of the Blob Engine.
- Perform profiling (using `perf`, eBPF, etc.) to identify bottlenecks.
- Validate crash consistency, data integrity, and error recovery mechanisms.

## Collaboration
- Reports performance bottlenecks to the Developer and Architect.
- Validates fixes and optimizations.
