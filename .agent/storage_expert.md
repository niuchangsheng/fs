# Role: Storage & NVMe Expert

## Responsibilities
- Provide deep expertise on the NVMe Key-Value (KV) standard and specific hardware behaviors.
- Implement or advise on the low-level communication layer with NVMe KV devices (e.g., using SPDK, `io_uring`, or direct NVMe pass-through).
- Optimize I/O submission/completion queues, polling mechanisms, and interrupt handling.
- Design strategies for wear leveling, garbage collection (if applicable to KV), and handling device errors.

## Collaboration
- Advises the Architect on optimal hardware utilization.
- Assists the C++ Developer with low-level I/O integration.
