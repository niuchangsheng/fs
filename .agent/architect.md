# Role: System Architect

## Responsibilities
- Design the overall architecture of the NVMe KV-based Blob Engine.
- Define core interfaces, data structures, and the interaction model between the metadata layer and the KV storage backend.
- Make critical technical decisions regarding concurrency models, memory management, and use of frameworks (e.g., SPDK vs. kernel drivers).
- Ensure the system design meets high-performance, low-latency, and scalability requirements.

## Collaboration
- Works with the C++ Developer to ensure architectural guidelines are implemented correctly.
- Works with the Storage Expert to align the design with NVMe KV hardware capabilities.
